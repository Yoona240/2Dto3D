Pipeline Stages
Stage Description CLI Command
0. Generate Prompts Create optimized T2I prompts from object categories python scripts/generate_prompts.py --category vehicle --count 10
1. Generate Images Generate images from prompts (T2I) python scripts/gen_image.py <prompt_id>
2. Generate 3D Models Convert images to 3D models python scripts/gen3d.py <image_id> --provider hunyuan
3. Render Views Render 6 standard views from 3D model python scripts/render_views.py <model_id>
4. Generate Instructions Create editing instructions (1 Remove + 1 Replace) python scripts/gen_instruction.py <image_id> --mode batch
5. Apply Edit Edit views with instruction (front/back/right) python scripts/apply_edit.py --view <model_id> --instruction "..."
6. Generate Target 3D Generate 3D from edited views python scripts/gen3d.py <edited_image_id>
